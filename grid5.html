Diffrence Between Justify-Items And Justify-Self:

Justify-Self:

The justify-self property aligns a grid item within its grid cell in the inline direction, For this property to have any alignment effect, the grid item need available space around itself in the inline direction

Justify-Items:

The justify-items property is set on the grid container to give child elements (grid items) alignment in the inline direction,For this property to have any alignment effect, the grid items need available space around themselves in the inline direction.
